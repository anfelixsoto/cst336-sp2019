What is AI?

Artificial Intelligence, or Machine Intelligence, is defined to be any device that learns its enivornment so it adapt to achieve a certain goals.


AI can also be defined as any task that can be done by a machine. If a human carried out that task, people would say the there had to be some use of intelligence to complete the task.

Current AI Systems

Smartphones

An example has to be "smart assisstants", these is just software to allow the user to perform task with simply their voice. Another example has to be Apple's "Animoji", this system uses 3D facial-mapping to allow them to move in real time.

Self-Driving Cars

This has to be the biggest part of AI technology. There are companies like Tesla, that have created vehicles that can adpat to current situations. The car is able to move out of danger situation if the driver does not response to it fast enough. Also, the car is equipped with senors and cameras to allow the car to drive itself! All this is using AI technology.
